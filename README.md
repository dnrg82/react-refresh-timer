# react-refresh-timer
react-refresh timer is a small package to add an auto-refresh component to your react site.

This timer will call your user-defined function, in which you can update only the react components you wish to be updated.
Note: this does NOT auto-refresh the browser page, it only refreshes the react components that you wish to refresh.
